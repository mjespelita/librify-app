Dashboard

Upon successful login, users are directed to the Dashboard, which provides a graphical summary of key item statuses within the system. The dashboard currently includes the following graphs:

  • In-Use Items graph. This bar graph displays the total count of items currently in use. It provides a clear visual representation of active inventory usage.

  • Damaged Items graph. This bar graph shows the total count of items that have been reported as damaged. It helps track asset condition and identify trends in equipment failure or misuse.

  • Items Overview graph presents a comprehensive summary of item distribution across different statuses. This graph includes both percentage and actual count values to provide a clear understanding of inventory allocation. It covers the following categories:
    • On-Site Items: Represents the percentage and count of items currently deployed or in use at various locations.

    • Damaged Items: Indicates the percentage and count of items that have been reported as damaged.

    • Warehouse Items: Shows the percentage and count of items that are stored and available in the warehouse.


  • In addition to the graphical insights, the dashboard also includes six summary boxes that provide quick, high-level metrics essential for operational awareness. Each box displays a key data point with a brief label and corresponding count.
    • Total Sites: Displays the total number of active deployment locations or project sites currently being managed within the system. This metric helps gauge the operational reach and scope of the organization.

    • Total Technicians: Indicates the number of technicians registered in the system. This provides a quick overview of available human resources responsible for installation, maintenance, and support tasks.

    • Item Types: Shows the total number of distinct item types currently being tracked (e.g., routers, modems, switches, etc.). This metric helps in understanding the diversity of assets in use across all operations.

    • Total Warehouse Items: Represents the total number of items stored and available in the warehouse. These items are in inventory but not yet deployed.

    • Total On-Site Items: Reflects the total number of items currently deployed or installed at various sites. This helps track how much of the inventory is actively in use.

    • Total Damaged Items: Displays the total number of items marked as damaged. This is critical for monitoring asset condition and planning for repairs or replacements.


  • At the bottom of the dashboard, users will find the Sites Overview section. This area provides a list of all registered sites, each displayed alongside its corresponding Google Maps location for easy geographic reference.

    Each site entry includes a “View Details” button. When clicked, this button reveals comprehensive information about the selected site, including:
    • Onsite Items: A detailed list of all items currently deployed at the site.

    • Assigned Technicians: Information on the technicians responsible for operations or maintenance at the location.

    • Damaged Items: A breakdown of any items at the site that have been reported as damaged.


Items

The Items page provides a detailed inventory of all assets currently managed within the system. Each item entry includes comprehensive information to help users track and manage equipment efficiently.

The following fields are displayed for each item:

  • Item ID: A system-generated unique identifier assigned automatically to each item for accurate tracking.

  • Name: The name of the item (e.g., Cisco Router, TP-LINK Switch, Modem).

  • Model: The specific model number or designation provided by the manufacturer.

  • Brand: The brand or manufacturer name associated with the item.

  • Type: The category or classification of the item (e.g., Router, Switch, etc.).

  • Description: A brief description of the item, outlining its purpose or features.

  • Total Overall Quantity: The total number of units recorded for the item across all statuses.

  • In Warehouse Quantity: The number of units currently stored in the warehouse and available for future deployment.

  • On-Site Quantity: The number of units actively deployed or installed at various sites.

  • Damaged Quantity: The number of units that have been marked as damaged and are no longer in usable condition.

  • Unit: The unit of measurement for the item (e.g., piece, meters, etc).

    • This page enables users to monitor inventory levels, manage stock distribution, and make informed decisions regarding asset utilization and replenishment.
